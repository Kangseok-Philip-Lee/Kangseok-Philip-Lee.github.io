블로그 글을 작성할 때 사용하는 Markdown 문법을 정리했습니다.

## 제목 (Headings)

```markdown
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
```

## 텍스트 스타일

```markdown
**굵게**
*기울임*
~~취소선~~
**_굵은 기울임_**
```

**굵게**, *기울임*, ~~취소선~~, **_굵은 기울임_**

## 목록

### 순서 없는 목록

```markdown
- 항목 1
- 항목 2
  - 하위 항목
  - 하위 항목
```

- 항목 1
- 항목 2
  - 하위 항목
  - 하위 항목

### 순서 있는 목록

```markdown
1. 첫 번째
2. 두 번째
3. 세 번째
```

1. 첫 번째
2. 두 번째
3. 세 번째

## 링크와 이미지

```markdown
[링크 텍스트](https://example.com)

![이미지 설명](/assets/images/example.jpg)
```

## 인용문

```markdown
> 인용문입니다.
> 여러 줄도 가능합니다.
```

> 인용문입니다.
> 여러 줄도 가능합니다.

## 코드

인라인 코드는 백틱으로 감쌉니다: `print("Hello")`

코드 블록은 백틱 세 개로 감싸고 언어를 지정합니다:

````markdown
```python
def hello():
    print("Hello, World!")
```
````

```python
def hello():
    print("Hello, World!")
```

## 수평선


## 체크박스

```markdown
- [x] 완료된 항목
- [ ] 미완료 항목
```

- [x] 완료된 항목
- [ ] 미완료 항목

## 각주

```markdown
이것은 각주가 있는 문장입니다[^1].

[^1]: 각주 내용입니다.
```

## 정리

위 문법만 알면 블로그 포스트를 작성하는 데 충분합니다. 자주 사용하면서 익숙해지면 빠르게 글을 쓸 수 있습니다.
